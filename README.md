# BoltDocs

> A minimal, fast, and highly customizable documentation framework powered by React and Vite.

## Features

- **File-Routing**: Automatically generate routes based on your directory structure.
- **Internationalization (i18n)**: Out-of-the-box support for multiple languages.
- **Documentation Versioning**: Manage multiple versions of your docs easily.
- **Light/Dark Mode**: Native dark mode with system preference detection and persistence.
- **UI & MDX Components**: Rich MDX components (cards, warnings, buttons, tabs, etc.).
- **Plugin System**: Extend functionality hooking into Vite's lifecycle.
- **Optimized for Performance**: Built-in code splitting, lazy loading, and asset optimization.
- **Monorepo Ready**: Designed to work seamlessly in monorepo environments using `pnpm` workspaces and `turbo`.

## Quick Start

```bash
# Clone the repository
git clone https://github.com/jesusalcaladev/boltdocs.git
cd boltdocs

# Install dependencies
pnpm install

# Start the development server
pnpm run dev
```
