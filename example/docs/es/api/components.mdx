# Components

Litedocs provides a set of built-in components to help you write richer documentation.

## PackageManagerTabs

The `PackageManagerTabs` component allows you to show an installation or execution command across multiple package managers (npm, yarn, pnpm, bun, deno) in a convenient, tabbed interface.

### Installation Example

Provide the `command="install"` prop and the package name in `pkg`:

```mdx
<PackageManagerTabs command="install" pkg="react" />
```

<PackageManagerTabs command="install" pkg="react" />


### Project Creation Example

You can also use it for `create` or `init` commands:

```mdx
<PackageManagerTabs command="create" pkg="vite@latest" />
```

<PackageManagerTabs command="create" pkg="vite@latest" />


### Run Example

It also supports run scripts:

```mdx
<PackageManagerTabs command="run" pkg="dev" />
```

<PackageManagerTabs command="run" pkg="dev" />
