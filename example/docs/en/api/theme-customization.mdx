# Theme Customization

Litedocs is designed to be highly customizable, utilizing a robust system of CSS variables. This allows you to easily tweak colors, typography, sizing, and more without writing complex CSS overrides.

## Light and Dark Modes

Litedocs fully supports light and dark themes natively. The theme is controlled by the `data-theme` attribute on any parent container or the `<html>` element.

By default, Litedocs uses the Dark theme. To activate the built-in Light theme, you can add `data-theme="light"` to your `<html>` element or `.theme-light` appropriately.

## Customizing CSS Variables

To build your own theme, simply target the `:root` pseudo-class (for the default dark mode) or `[data-theme="light"]` (for light mode) in your global styles.css and override the necessary variables.

### Example: A Custom Accent Color

Let's change the default purple accent to a vibrant green:

```css
:root {
  /* Main vibrant green */
  --ld-color-primary: #10b981;
  /* Darker version for hover */
  --ld-color-primary-hover: #059669;
  /* Transparent versions for backgrounds and glows */
  --ld-color-primary-muted: rgba(16, 185, 129, 0.10);
  --ld-color-primary-glow: rgba(16, 185, 129, 0.25);
  /* The text color inside primary buttons */
  --ld-color-primary-text: #ffffff;
}
```

### Color Variables

Here is the complete palette of background and text colors you can configure:

```css
:root {
  /* Background Colors */
  --ld-bg-main: #0a0a0f;     /* Main background of the page */
  --ld-bg-soft: #0f0f18;     /* Slightly lighter backgrounds (Sidebar, Footer) */
  --ld-bg-mute: #141420;     /* Muted backgrounds (Search inputs) */
  
  /* Foreground Colors */
  --ld-text-main: #e4e4ed;   /* Primary text color */
  --ld-text-muted: #9d9db5;  /* Secondary / paragraph text */
  --ld-text-dim: #6b6b85;    /* Dimmed text (metadata, small details) */
  
  /* Border Colors */
  --ld-border-subtle: rgba(255, 255, 255, 0.06);
  --ld-border-strong: rgba(255, 255, 255, 0.12);
}
```

### Typography

To change the core fonts used across your documentation, define the following variables:

```css
:root {
  --ld-font-sans: 'Inter', system-ui, sans-serif;
  --ld-font-mono: 'Fira Code', monospace;
}
```

*(Ensure you also import your desired web fonts via `@import` or `<link>` tags).*

### Layout Dimensions

You can adjust the sizing of the main components (Navbar, Sidebar) or the maximum width of the reading area:

```css
:root {
  --ld-navbar-height: 4rem;
  --ld-sidebar-width: 16rem;
  --ld-toc-width: 14rem;
  --ld-content-max-width: 900px;
}
```
