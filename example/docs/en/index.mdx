# Welcome to Litedocs

This is the homepage of our example documentation site built with `litedocs`.
It uses MDX natively, so you can embed **React Components**!

## Features

- File-system based routing
- Auto-generated sidebar with groups
- Syntax highlighting with Shiki
- "On This Page" table of contents
- Custom landing page support
- Hot Module Replacement (HMR)

```typescript
// All configuration is done in vite.config.ts
import litedocs from 'litedocs';

export default defineConfig({
  plugins: [
    litedocs({
      docsDir: 'docs',
      homePage: './src/HomePage.tsx',
    })
  ],
});
```
