# Getting Started

Welcome to Litedocs! This guide will walk you through setting up your first documentation site.

## Installation

First, install the required dependencies:

```bash
npm install litedocs react react-dom react-router-dom
```

Make sure you have **Node.js 18+** and **Vite 4+** installed.

## Project Setup

Create a new Vite project and add litedocs as a plugin:

```typescript
// vite.config.ts
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import litedocs from 'litedocs';

export default defineConfig({
  plugins: [
    react(),
    litedocs({ docsDir: 'docs' })
  ],
});
```

### Folder Structure

Your project should look like this:

```
my-docs/
├── docs/
│   ├── index.mdx
│   ├── getting-started.mdx
│   └── api/
│       ├── index.mdx
│       └── hooks.mdx
├── src/
│   └── main.tsx
├── index.html
└── vite.config.ts
```

### Entry Point

Your `main.tsx` should use `createLitedocsApp`:

```tsx
import { createLitedocsApp } from 'litedocs/client'
import routes from 'virtual:litedocs-routes'
import config from 'virtual:litedocs-config'
import 'litedocs/style.css'

createLitedocsApp({
  target: '#root',
  routes,
  config,
  modules: import.meta.glob('/docs/**/*.{md,mdx}'),
  hot: import.meta.hot,
})
```

## Writing Documentation

Each `.mdx` file in your `docs/` folder automatically becomes a page.

### Frontmatter

Use YAML frontmatter to configure each page:

```yaml
---
title: My Page Title
sidebarPosition: 1
---
```

### Route Groups

Create folders inside `docs/` to group pages in the sidebar:

```yaml
---
title: API Overview
groupTitle: API Reference
groupPosition: 10
---
```

The `groupTitle` and `groupPosition` are defined in the folder's `index.mdx`.

## Syntax Highlighting

Code blocks are automatically highlighted using **Shiki** with the `vitesse-dark` theme:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
  return { greeting: `Hi ${name}`, timestamp: Date.now() };
}
```

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)
```

## Customization

### Theme Config

You can customize the navbar and site title via configuration:

```javascript
// docs.config.js
export default {
  themeConfig: {
    title: 'My Docs',
    navbar: [
      { text: 'Home', link: '/' },
      { text: 'API', link: '/api' },
    ],
  },
};
```

### CSS Variables

Override CSS variables to change the look and feel:

```css
:root {
  --litedocs-primary: #10b981;
  --litedocs-bg: #fafafa;
  --litedocs-sidebar-bg: #f0fdf4;
}
```

## Deployment

Build your docs for production:

```bash
npm run build
```

The output will be in `dist/`, ready for static hosting on **Netlify**, **Vercel**, or **GitHub Pages**.

### Static Hosting

Upload the `dist/` folder to any static file server. No server-side rendering required.

### Custom Domain

Point your domain's DNS to the hosting provider and configure the base path if needed in `vite.config.ts`.
