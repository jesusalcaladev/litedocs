# MDX Components

Because LiteDocs leverages React and MDX, standard Markdown elements are automatically enhanced with beautifully styled React components underneath the hood. 

Moreover, LiteDocs ships with several utility UI components you can drop directly into your Markdown! 

Explore the individual component pages in the sidebar to learn more about how to use each one:

- **Admonitions**: Callouts for info, warnings, and errors.
- **Button**: Versatile button and link component.
- **Badge**: Inline status indicators.
- **Cards**: Grid-based content blocks.
- **Tabs**: Tabbed content organization.
- **List**: Enhanced feature lists with icons.
- **FileTree**: Visual file structure representation.
- **Table**: Responsive tables with sorting and pagination.
- **Custom Components**: How to integrate your own React components.
