# Button

A versatile button/link component for use in MDX pages and the home page.

**Example:**

<div style={{ display: 'flex', gap: '1rem', flexWrap: 'wrap' }}>
  <Button variant="primary" size="md">
    Get Started
  </Button>

  <Button variant="secondary" size="md">
    Click Me
  </Button>
  
  <Button variant="outline" size="md">
    Outline
  </Button>
</div>

```tsx
<Button variant="primary" size="md">
  Get Started
</Button>

<Button variant="secondary" size="md" >
  Click Me
</Button>

<Button variant="outline" size="md">
  Outline
</Button>
```

**Props (`ButtonProps`)**

<Field name="variant" type='"primary" | "secondary" | "outline" | "ghost"' defaultValue='"primary"'>
  The visual style variant of the button.
</Field>

<Field name="size" type='"sm" | "md" | "lg"' defaultValue='"md"'>
  The sizing scale of the button.
</Field>

<Field name="href" type="string">
  If provided, the element renders as an `<a>` tag instead of a `<button>`.
</Field>

_(Standard anchor and button HTML attributes are also supported)_
