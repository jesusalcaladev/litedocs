# List

An enhanced list component with specific icon variants, perfect for feature listings or checklists.

**Example:**

<div style={{ display: 'flex', gap: '4rem', flexWrap: 'wrap' }}>
  <div>
    **Checked Variant:**
    <List variant="checked">
      <li>File-system routing</li>
      <li>MDX support</li>
      <li>Syntax highlighting</li>
    </List>
  </div>
  <div>
    **Arrow Variant:**
    <List variant="arrow">
      <li>First step</li>
      <li>Second step</li>
      <li>Third step</li>
    </List>
  </div>
</div>

```tsx
<List variant="checked">
  <li>File-system routing</li>
  <li>MDX support</li>
  <li>Syntax highlighting</li>
</List>

<List variant="arrow">
  <li>First step</li>
  <li>Second step</li>
  <li>Third step</li>
</List>
```

**Props (`ListProps`)**

<Field name="variant" type='"default" | "checked" | "arrow"' defaultValue='"default"'>
  The visual icon injected before every list item.
</Field>

_(Standard `<ul>` HTML attributes are also supported)_
