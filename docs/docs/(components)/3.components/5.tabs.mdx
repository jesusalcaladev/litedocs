# Tabs

A component used to categorize content and let the user conditionally render different views, like multiple code snippets for different package managers.

**Example:**

<Tabs defaultIndex={0}>
  <Tab label="npm">
    npm install boltdocs
  </Tab>
  <Tab label="pnpm">
    pnpm add boltdocs
  </Tab>
  <Tab label="bun">
    bun add boltdocs
  </Tab>
</Tabs>

````tsx
<Tabs defaultIndex={0}>
  <Tab label="npm">
    npm install boltdocs
  </Tab>
  <Tab label="pnpm">
    pnpm add boltdocs
  </Tab>
  <Tab label="bun">
    bun add boltdocs
  </Tab>
</Tabs>
````

**`<Tabs>` Props (`TabsProps`)**

<Field name="defaultIndex" type="number" defaultValue="0">
  The index (0-based) of the tab that is open by default.
</Field>

**`<Tab>` Props (`TabProps`)**

<Field name="label" type="string" required>
  The text shown in the tab bar trigger.
</Field>
