# Admonitions (Callouts)

Admonitions highlight important information. Using them is as simple as adding an `<Admonition>` tag or using the markdown syntax if you have remark plugins configured.

**Using `<Admonition>`**

<Admonition type="info" title="Hey there!">
  This is a helpful note for the reader.
</Admonition>

<Admonition type="warning" title="Warning">
  Pay attention to this warning block.
</Admonition>

<Admonition type="danger" title="Danger">
  This is a destructive action notice.
</Admonition>

```tsx
<Admonition type="info" title="Hey there!">
  This is a helpful note for the reader.
</Admonition>

<Admonition type="warning" title="Warning">
  Pay attention to this warning block.
</Admonition>

<Admonition type="danger" title="Danger">
  This is a destructive action notice.
</Admonition>
```

**Props (`AdmonitionProps`)**

<Field name="type" type='"info" | "warning" | "danger"' defaultValue='"info"'>
  The visual variant of the callout.
</Field>

<Field name="title" type="string">
  Optional. The title displayed at the top of the admonition.
</Field>

> **Note**: You can also use explicit tags like `<Note>`, `<Tip>`, `<Warning>`, `<Danger>`, or `<InfoBox>` as shorthands.
