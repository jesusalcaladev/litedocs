# FileTree

A component to display folder and file structures using standard Markdown lists. It automatically assigns the correct icons based on the file extension or folder syntax.

**Example:**

<FileTree>
- astro.config.mjs
- package.json
- src/
  - components/
    - Header.astro
    - Title.astro
  - pages/
</FileTree>
