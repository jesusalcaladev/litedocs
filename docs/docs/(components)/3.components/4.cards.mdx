# Cards

Cards are used to display features, directories, or distinct pieces of information in a grid. They are wrapped in a `<Cards>` container.

**Example:**

<Cards cols={3}>
  <Card title="Fast" icon="⚡" href="/docs/getting-started/installation">
    Instant HMR thanks to Vite.
  </Card>
  <Card title="Simple" icon="📁">
    File-system based routing.
  </Card>
  <Card title="Beautiful" icon="✨">
    Built-in modern MDX components.
  </Card>
</Cards>

