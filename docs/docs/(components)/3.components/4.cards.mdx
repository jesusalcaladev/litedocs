# Cards

Cards are used to display features, directories, or distinct pieces of information in a grid. They are wrapped in a `<Cards>` container.

**Example:**

<Cards cols={3}>
  <Card title="Fast" icon="⚡" href="/docs/getting-started/installation">
    Instant HMR thanks to Vite.
  </Card>
  <Card title="Simple" icon="📁">
    File-system based routing.
  </Card>
  <Card title="Beautiful" icon="✨">
    Built-in modern MDX components.
  </Card>
</Cards>

```tsx
<Cards cols={3}>
  <Card title="Fast" icon="⚡" href="/docs/getting-started/installation">
    Instant HMR thanks to Vite.
  </Card>
  <Card title="Simple" icon="📁">
    File-system based routing.
  </Card>
  <Card title="Beautiful" icon="✨">
    Built-in modern MDX components.
  </Card>
</Cards>
```

**`<Cards>` Props (`CardsProps`)**

<Field name="cols" type="1 | 2 | 3 | 4" defaultValue="3">
  Number of columns to display in the grid.
</Field>

**`<Card>` Props (`CardProps`)**

<Field name="title" type="string">
  The main heading of the card.
</Field>

<Field name="icon" type="ReactNode | string">
  Emoji or icon displayed above the title.
</Field>

<Field name="href" type="string">
  If provided, the entire card becomes a clickable link.
</Field>
