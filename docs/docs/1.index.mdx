# What is Boltdocs?

Boltdocs is a minimal, blazing-fast, and highly customizable documentation framework built on top of **React** and **Vite**. Our goal is to provide a developer experience that is as frictionless as possible out of the box, while giving you all the escape hatches you need to build stunning, interactive documentation sites.

## Why Boltdocs?

Most documentation tools are either too complex (requiring you to learn new templating languages) or too rigid (making it hard to add custom UI or overrides). Boltdocs is different:
- **It’s just React + MDX**. You have full access to React components inside your Markdown files.
- **Powered by Vite**. So your development build starts instantly and hot module replacement (HMR) is extremely fast.
- **Built-in Features**. Full support for i18n, versions, dark mode, code snippets, search, and more, all ready from day one.

## Core Features

- ⚡️ **Fast by Default**: Preloading, lazy loading, and asset optimizations are heavily utilized.
- 📁 **File-System Routing**: Create a markdown file, and you get a route.
- 🎨 **Theme-ability**: A dedicated variables file (`custom.css`) to tweak the default pristine clean theme effortlessly.
- 🌍 **Internationalization (i18n)**: Out-of-the-box support via nested directories (`docs/es/`, `docs/en/`).
- 📚 **Versioning**: Maintain `v1` and `v2` easily using standard directory structures.
- 🧩 **Plugins System**: Tie into Vite's lifecycle and extend the framework's behavior.
- 💅 **UI Components**: Comes with built-in MDX components like `Card`, `Admonition`, Tabs, and a live React `Playground`.
